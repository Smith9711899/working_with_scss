Design System

This is a reusable UI system for consistent design and development.

Made By: Jonathan Smith


Design Tokens


Colors

$primary-color-700: #889AD0

This is used for things such as primary buttons, links...

$success-color: #28a745

This is used for success states (e.g. selected radio)

Spacing

$space-2: 8px or 0.5rem

This is used for making space between something like a checkbox and text

$space-4: 16px or 1rem

This is used to make space for something like a heading and text

Typography

Font: Inter, sans-serif

Base size: 16px

line height and font weight has been give per-sets to keep everything uniform
